## Quick Start

```sh
npm install
npm start
```
This will start the development environment for the electron app.

## Build 

### Windows
```sh
npm run package-win
```

### MacOS
```sh
npm run package-mac
```

### Linux
```sh
npm run package-linux
```
<br>
Packaged app is stored in the ./dist folder.
